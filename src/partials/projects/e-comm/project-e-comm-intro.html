Аудитория

На сайты МТС приходят пользователи из контекстной рекламы. Среди них есть целевые, уже готовые подключиться, и нецелевые с запросами уровня техподдержки

Цель

Увеличить качество входящего трафика, сократить время обработки обращений в колл-центре и не сохранить целевую аудиторию

Задачи

  • Спроектировать новый одностраничный сайт для проверки гипотез
  • Убрать телефоны с сайта, чтобы отсеять звонки в техподдержку
  • Фильтровать пользователей на ЦА и не ЦА
  • Реализовать функцию, которая поможет узнать, что пользователь хочет, новое подключение или апдейт тарифа
  • Если подключение недоступно, предложить другие варианты
  • Передавать оператору уже готовые структурированные данные для обработки
