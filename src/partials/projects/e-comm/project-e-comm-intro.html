Аудитория

На сайты МТС приходят пользователи из контекстной рекламы. Среди них есть целевые, уже готовые подключиться, и нецелевые с запросами уровня техподдержки

Цель

Увеличить качество входящего трафика и сократить время обработки обращений в колл-центре за счет фильтрации пользователей и автоматизации сбора данных

Задачи

  • Спроектировать новый одностраничный сайт для проверки гипотез
  • Убрать все телефоны с сайта, чтобы отсеять звонки в техподдержку
  • Реализовать механизмы фильтрации целевой аудитории
  • Минимизировать работу оператора, пользователь сам выбирает тариф, оборудование и услуги
  • Передавать оператору уже готовые структурированные данные для обработки
