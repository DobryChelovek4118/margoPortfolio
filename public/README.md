# public/

Статические файлы — отдаются с корня сайта (`/`).

## Что сюда положить

### Favicon
| Файл | Размер | Формат |
|---|---|---|
| `favicon.ico` | 32×32 px | ICO |
| `favicon.svg` | любой | SVG |
| `favicon-32x32.png` | 32×32 px | PNG |
| `favicon-16x16.png` | 16×16 px | PNG |
| `apple-touch-icon.png` | 180×180 px | PNG |

### OG-изображение
| Файл | Размер | Формат |
|---|---|---|
| `og-image.jpg` | 1200×630 px | JPG, до 300 KB |

## Как сделать favicon

1. Подготовь SVG или PNG 512×512 px с логотипом/инициалами
2. Сгенерируй все форматы: https://realfavicongenerator.net
3. Положи файлы сюда
